# Post-it-Board Frontend

A React + TypeScript + Vite application for the post-it-board.

## Getting Started

### Prerequisites

-   Node.js (v16 or higher)
-   npm or yarn

### Installation

```bash
npm install
```

### Running the App

```bash
npm run dev
```

The app will be available at `http://localhost:5173`

### Running the Linter

```bash
npm run lint
```

## Server Connection

This frontend connects to a server running on `http://localhost:3000`. Make sure the server is running before using the app.

For server setup instructions, see the [server README](../server/README.md).
